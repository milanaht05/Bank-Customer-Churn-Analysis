# Bank Customer Churn Analysis

## Project Overview
This project analyzes customer churn in the banking sector using Exploratory Data Analysis (EDA).

## Dataset Information
- Records: 10,000
- Features: 14
- Target Variable: Exited

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Older customers are more likely to churn.
- Inactive customers have higher churn rates.
- Customers with higher balances show higher churn tendencies.
- Customers using more products are less likely to leave.

## Business Recommendations
- Increase customer engagement.
- Offer loyalty programs for older customers.
- Provide personalized services for high-value customers.
- Promote cross-selling of banking products.

## Conclusion
The analysis identified important factors influencing customer churn and provides actionable recommendations to improve customer retention.
